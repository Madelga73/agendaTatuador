Ejecución

en powershell dentro de la carpeta correspondiente ejecutar el comando: 
docker compose up --build
, pero antes verificar que docker este activo.

ya teniendo buildeado el proyecto verificar que este prendido...
abrir el postman para asi probar los endpoints y poder usar de una manera correcta nuestro proyecto
